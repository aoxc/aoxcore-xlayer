## 💠 Overview

**AOXCORE** is a high-performance Neural Operating System designed to bridge the gap between **Generative AI** and **Blockchain Infrastructure**. Integrated with the **Gemini Sentinel Engine**, it provides real-time security auditing, automated smart contract synthesis, and a neural-styled management interface for decentralized protocols.

> **View Live Instance (AI Studio):** [Neural OS Dashboard](https://ai.studio/apps/f76f5af7-5fa1-4960-8e9e-e328cf255d14)

---

## 🛠️ Tech Stack

* **Kernel:** Foundry (Solidity) - Advanced Smart Contract Suite.
* **Neural Engine:** Google Gemini AI Core - Automated Threat Detection.
* **OS Interface:** React 19 + Vite + Tailwind CSS (V4).
* **Networking:** Viem & Wagmi - High-fidelity blockchain provider integration.

## 🚀 Deployment & Local Setup

### Prerequisites
-   **Node.js** (v20+ recommended)
-   **Yarn** or **NPM**
-   **Foundry** (For kernel/contract operations)

### 1. Initialize Neural Environment
Clone the repository and install the protocol dependencies:
```bash
# Clone the core
git clone [https://github.com/aoxc/AOXCORE.git](https://github.com/aoxc/AOXCORE.git)
cd AOXCORE/frontend

# Install OS dependencies
yarn install  # or npm install
