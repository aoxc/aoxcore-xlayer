## 💠 Overview

**AOXCORE** is a high-performance Neural Operating System designed to bridge the gap between **Artificial Intelligence** and **Blockchain Infrastructure**. 

By leveraging the **Gemini Sentinel Engine**, it provides a decentralized environment for real-time security auditing, neural-styled protocol management, and automated smart contract synthesis.

---

## 🛠️ Tech Stack

* **Core Kernel:** Foundry (Solidity) - Advanced Smart Contract Suite.
* **Neural Engine:** Google Gemini AI - Automated Threat Detection.
* **Interface:** React 19 + Vite + Tailwind CSS (V4).
* **Web3 Layer:** Viem & Wagmi - High-fidelity blockchain integration.
